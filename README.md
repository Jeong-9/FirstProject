# FirstProject

## 1. 프로젝트 개요
WinForms와 MySQL을 기반으로 제작한 일정관리 프로그램입니다.  
사용자는 회원가입 및 로그인을 통해 서비스를 이용할 수 있으며,  
아이디 찾기, 비밀번호 찾기, 임시 비밀번호 이메일 발송, 비밀번호 변경 기능까지 구현하였습니다.

## 2. 프로젝트 배경
기존의 단순한 일정관리 프로그램은 일정 입력 기능에 집중되어 있는 경우가 많아  
실제 서비스처럼 사용자 계정 관리 기능이 부족하다고 생각했습니다.  
따라서 본 프로젝트에서는 일정관리 기능과 함께 회원관리 기능까지 포함한  
보다 완성도 있는 프로그램을 구현하고자 하였습니다.

## 3. 프로젝트 목적
- 사용자 계정 기반의 일정관리 프로그램 구현
- 회원가입 및 로그인 기능을 통한 사용자 인증
- 아이디 찾기 및 비밀번호 찾기를 통한 계정 복구 기능 제공
- 실제 이메일로 임시 비밀번호를 발송하는 기능 구현
- 비밀번호 변경 기능을 통한 보안성과 편의성 향상

## 4. 주요 기능
### 로그인
- 아이디와 비밀번호를 입력하여 사용자 인증 수행
- 로그인 성공 시 사용자 세션 유지

### 회원가입
- 사용자 정보 입력 후 DB에 회원 정보 저장
- 입력값 확인 및 중복 검사 수행

### 아이디 찾기
- 가입 시 입력한 사용자 정보를 통해 아이디 조회 가능

### 비밀번호 찾기
- 가입된 이메일 확인 후 임시 비밀번호 생성
- 실제 이메일로 임시 비밀번호 발송

### 비밀번호 변경
- 임시 비밀번호 또는 기존 비밀번호 확인 후 새 비밀번호로 변경 가능

## 5. 개발 환경
- Language : C#
- Framework : WinForms
- Database : MySQL
- IDE : Visual Studio
- Version Control : GitHub

## 6. 핵심 클래스
- JoinMember  
  회원가입, 로그인, 아이디 찾기, 비밀번호 찾기, 비밀번호 변경 관련 기능 담당

- UserSession  
  로그인한 사용자 정보를 프로그램 전역에서 유지

## 7. 기대 효과
- 사용자 인증이 포함된 일정관리 프로그램 구현 경험 축적
- 데이터베이스와 UI, 이메일 발송 기능을 연동한 프로젝트 완성
- 실제 프로그램 형태에 가까운 회원관리 시스템 구현

## 8. 향후 개선점
- 비밀번호 암호화 저장 적용
- 이메일 인증번호 방식 추가
- UI 디자인 개선
- 예외 처리 및 보안 강화
